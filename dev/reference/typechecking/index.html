DiffPrivacyInference.simplify_constraints_lose_generality — Method

simplifyconstraintslose_generality() :: TC

Simplify as many constraints as possible, then assume all free type variables involved in DMTypeOps are non-constant and simplify again.

This worsens the sensitivity, as non-constant types yield larger sensitivity penalty. It is hence legal to do, but should only be done if no other simplification is possible for best results.
